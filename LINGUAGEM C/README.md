# Atividade de Programação em C

Este repositório contém a atividade realizada para a disciplina de programação da faculdade.

## Descrição
O código desenvolvido nesta atividade tem como objetivo praticar conceitos básicos de programação em linguagem C, como:
- Estruturas de decisão
- Laços de repetição
- Manipulação de variáveis
- Entrada e saída de dados

## Tecnologias utilizadas
- Linguagem C
- Compilador GCC (GNU Compiler Collection)

## Como compilar e executar
Para compilar o programa:
```bash
gcc atividade_operadores.c -o programa
